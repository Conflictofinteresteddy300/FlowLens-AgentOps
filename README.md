## 🎯 About this software

## 🛠 Using the interface

Once the application opens, you see a clean dashboard. This view lists all active agents. Click on any agent name to view its recent activity. Each record shows a timeline of actions. You see exactly when a failure occurred. 

The software highlights errors in red text. Select any error to see the details behind the event. Use the replay button to watch the steps the agent followed before the error. This helps you identify the specific moment the process stopped.

## 📈 Key features

*   **Failure Attribution**: The software points to the exact part of your workflow that triggered an error.
*   **No-Key Replay**: You restart agent tasks to verify fixes without needing to maintain complex access keys.

**Will this tool slow down my agents?**
